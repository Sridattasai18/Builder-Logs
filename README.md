# Builder Log

A beautiful, local-first, single-page application for developers to document their projects, features, and ideas. Built purely with HTML, CSS, and JS — no servers, no databases, just a clean interface.

![Builder Log Interface](https://via.placeholder.com/800x450.png?text=Builder+Log+Dashboard)

## 🌟 Features

- **Local-First:** All data is saved directly in your browser's `localStorage`. True privacy by design.
- **Markdown Editor:** Native support for GitHub-flavored markdown with side-by-side live previews.
- **Card Dashboard:** See all your logs in a clean grid with tags, categories, and quick links.
- **Search & Filter:** Instantly filter logs by clicking tags or using the real-time search bar.
- **Tag Management System:** Autocomplete tags while writing, create new ones, and assign custom pastel colors.
- **Export & Import:** Export all your logs as a portable JSON backup file, or export individual logs as clean PDFs.
- **Zero Dependencies:** Runs from a simple HTML file. No npm, no node_modules, no build step.

## 🚀 Quick Start

Builder Log doesn't require any installation. Because it has zero dependencies, you can run it right from your file system.

1. Clone or download the repository.
2. Double click `index.html` to open it in your browser.
3. Start writing!

## 💾 Data Portability

Since Builder Log uses `localStorage`, your data stays in the browser profile where you open it.

- Use the **⬇ Export JSON** button to back up your logs.
- Use the **⬆ Import JSON** to restore them on another machine or browser.

## 🛠️ Usage Tips

- **Keyboard Shortcuts:** Use `Ctrl + S`/`Cmd + S` in the editor to quickly save. Press `Escape` to close the reader or editor.
- **External Links:** You can attach GitHub and Live Demo links directly to your project cards for quick access.
- **Chat Formatting:** Use blockquotes in Markdown for chat logs! Odd blockquotes become right-aligned bubbles (You), and even blockquotes become left-aligned bubbles (AI).
